# Portfolio Développeur Web — HTML / CSS / JavaScript

Un portfolio moderne et responsive développé avec **HTML**, **CSS** et **JavaScript** permettant de présenter un profil développeur, ses projets, ses compétences et ses contacts.

---

## 🚀 Fonctionnalités

- ✅ Design moderne et responsive
- ✅ Barre de navigation responsive
- ✅ Section présentation (Hero)
- ✅ Section compétences
- ✅ Section projets
- ✅ Animation au scroll
- ✅ Menu mobile toggle
- ✅ Effets hover modernes
- ✅ Formulaire de contact
- ✅ Footer professionnel
- ✅ JavaScript interactif
